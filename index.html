Projects

I design integrated technology using full-stack development and real-time embedded systems.

Project 01

Rear-Vision Camera

Built a real-time rear-vision system for Northwestern's 2026 solar car using a Pi Camera Module 3 Wide with CSI to HDMI extensions to achieve sub-40ms latency at 60 fps over 3 meters. My work also included benchmarking six platforms against automotive grade standards GMSL2 and FPD-Link III. Also developed an interactive HTML dashboard for performance data using chat.js

Raspberry Pi HTML Embedded Systems
Project 02

FPGA FOC Control

Implementing Field Oriented Control in Verilog HDL on an FPGA to maximize PWM switching frequency and reduce current ripple in lightweight motors. Benchmarking hardware accelerated motor control against commercial controllers and designing a custom PCB for the NUSolar Mitsuba M2096

Verilog HDL FPGA Altium PCB Design
Project 03

FPGA Beamforming Array

Building a FPGA beamforming microphone array with real-time spatial tracking that combines camera-based face tracking with FPGA signal processing to locate and follow a sound source in real time, coherently summing inputs into spatially focused audio.

FPGA DSP Computer Vision
